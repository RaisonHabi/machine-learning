## DataFrame 数据的保存和读取
### csv
df.to_csv 写入到 csv 文件  
pd.read_csv 读取 csv 文件  
示例：  
df.to_csv("xxx.csv")


### json
df.to_json 写入到 json 文件  
pd.read_json 读取 json 文件

### html
df.to_html 写入到 html 文件  
pd.read_html 读取 html 文件

### excel
df.to_excel 写入到 excel 文件  
pd.read_excel 读取 excel 文件
