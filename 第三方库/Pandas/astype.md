## astype
DataFrame.**astype**(self: ***~FrameOrSeries***, dtype, copy: bool = True, errors: str = 'raise')  

Cast a pandas object to a specified dtype dtype.

## 示例
详见参考  
> df.astype('int32').dtypes  
col1    int32  
col2    int32  
dtype: object
