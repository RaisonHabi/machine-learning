## argsort()  
**argsort(a, axis=-1, kind='quicksort', order=None)**  
Returns the indices that would sort an array.  
***argsort函数返回的是数组值从小到大的索引值***  
示例:  
> x = np.array([3, 1, 2])  
np.argsort(x)  
array([1, 2, 0])
