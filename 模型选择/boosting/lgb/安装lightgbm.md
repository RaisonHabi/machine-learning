## 简单方法
win和linux平台都可以通过pip install lightgbm安装最新的lightgbm了，现在版本是lightgbm-2.0.3。按照本教程安装的童鞋也可用命令pip install –upgrade lightgbm升级到lightgbm最新版本。
```
pip install lightgbm
```
安装tqdm库：
```
pip install tqdm
```
